# FizzBuzz
FizzBuzz is a .Net 5 Web API that taken in a Json formatted list of input and gives a result as:
 
Sample Input Format(Json): 
{
  "input": [
    "1",
    "2",
    "3",
    "5",
    "15",
    "ABC",
    "25"
  ]
}

Sample Output:
[
  "Divided 1 by 3",
  "Divided 1 by 5",
  "Divided 2 by 3",
  "Divided 2 by 5",
  "Fizz",
  "Buzz",
  "FizzBuzz",
  "Invalid Item",
  "Buzz"
]
